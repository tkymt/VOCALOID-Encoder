# VOCALOID Encoder
とにかく高音質に動画投稿したい、みんなのためのエンコーダー

## メリット
・こえが聞き取りやすい。
・ボーカロイド楽曲に最適です。
・ゆっくり音声などにも適しています。
・どんなキャラクターの音声ソフトウェアでも正しく動作します。
・ASMRなど、マイクで収録した音声なら、人間でも動物でも楽器でも、なんでもかんでも高音質かつ高品質になります。
・ニコニコ動画にもYouTubeにも投稿できるファイルがつくれる
・ボカコレやニコニコ動画でサムネイルの生成が、（H.264を使用しているため、H.265と比べて）はやい。
それらの効果によって、VOCALOID（無印）やVOCALOID2など、かつてのプロジェクトに何の変更も加えずに、そのままPCMでレンダリングすれば、それだけでリマスター盤になります。

## デメリット
いまのところ静画に音声がついた動画になります。
連番画像によるアニメは、つくれません。（コマンドラインオプションを改変すれば可能になります）
音声の音源に「PCMファイル形式」を用いますから、そのほかの音声ファイル形式よりも最大再生時間が比較的、短くなります。
たとえば、64bit float × 192kHzなら、だいたい65分に制限されます。
（そんなにもハイレゾなPCMにレンダリングしない！）と思われるかもしれませんが、なんとなく明記しておきます。
それでも、ニコニコ動画の投稿要件の6GB以下かつ6時間以内という制限に納まりますからチャレンジ精神を持って試みるのもよろしいかと思います！！
ノイズが含まれる音源だったらば、ノイズが目立ちます。ただ、目立つからといって目くじらを立てないよう、お願いします。

# つかいかた
それでは、つかいかたを説明します。
「input.wav」と「input.pfm」を準備し、「☆彡Run.bat」を実行してください。
そのあとは、黒い画面の指示に従ってください。
従った後は、「☆Play.bat」を実行すると、再生して最終確認してください。
最終確認のあとは、投稿してください。
※ただし、ここでは動画投稿の方法を説明しません。あしからず。

## より詳細なつかいかた
ひつようなもの：
📺　ニコニコ動画を視聴できるパソコンならなんでも。
📺　PCMファイル形式の音源を１つ以上。
📺　pfmファイル形式の画像を１つだけ。
📺　それから、チャレンジ精神。なぜならパソコンが熱くなり、フゥファ～～～🎶というような、冷却用のファンの音がします。それでも爆発はしませんのでご安心ください。

## 準備：ダウンロード編
いま、みている画面の、右上らへんの、緑色の［＜＞コード］を押してください。
押すとプルダウンメニューが現れます。
プルダウンメニューから［ＺＩＰファイルをダウンロード］を選択してください。
ダウンロード先は、「ダウンロード」または「デスクトップ」にしてください。
ダウンロードが完了したら、完了したファイルを選択して、［すべて展開］をしてください。
そうしないと、Windows Defenderにおいての書き込み権限を、与えなければならなくなります。

## 準備：音源編
PCM音源を用意してください。
音源のファイル名は、「input.wav」にしてください。
「input.wav」を「Run.bat」があるフォルダにコピーしてください。
大事なことなので、もう一度、小さな声で言います。
（かならず、コピー、に、して、ください。）

## 準備：MikuMikuDance ポーズとカメラ編
MikuMikuDanceで、キャラクターに対してポーズとカメラワークをつけて、まっ先に[Ctrlキー]を押しながら、[Sキー]を押します。
こうすることで、プロジェクトを「.pmm」ファイルで保存でき、さらにキーフレームが打ち込まれます。

## 準備：MikuMikuDance 画像ファイルに出力編
プロジェクトを保存したら、画像ファイルに出力します。
MikuMikuDanceの画面の左上にある［ファイル］を押してください。
押すと、プルダウンメニューが現れます。
プルダウンメニューから［画像ファイルに出力］を選択してください。
［画像ファイルに出力］を選択すると、あらたに「画像ファイル出力」画面があらわれます。
「画像ファイル出力」画面の下らへんにある「ファイルの種類(T):」の右隣にある［すべての対応フォーマット］を押してください。
［すべての対応フォーマット］を押すと、プルダウンメニューが現れます。
プルダウンメニューから［pfm files(*.pfm)］を選択します。

## 準備：MikuMikuDance 画像保存編
［pfm files(*.pfm)］を選択出来たら、「ファイル名(N):」の右隣の空の入力欄を［クリック］してください。
［クリック］できたら、「input」と［キーボードをタイプ］してください。
［タイプ］できたら、右下らへんにある［保存(S)］を［クリック］してください。
こうすると、「input.pfm」が出来上がります。
出来上がった、「input.pfm」を「Run.bat」があるフォルダにコピーしてください。

## 実行：エンコーダー
「☆彡Run.bat」を［ダブルクリック］または［選択してエンターキー］で実行してください。
あらわれた黒い画面の、書かれている指示に従ってください。

## 正しく動作したか、確認する方法
「☆Play.bat」を実行してください。
実行するには、「☆Play.bat」を実行してください。
実行すると、動画が再生されます。
動画を再生中に、ダブルクリックすると、映像の見た目の大きさがディスプレイにフィットします。
動画の再生を中止するには、［Escキー］を押してください。
また、最後まで再生し終えたら、［Ecsキー］を押してください。
